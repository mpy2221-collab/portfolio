# 영화 추천·리뷰 웹 서비스

유저가 영화를 추천(유저픽)하고 리뷰를 작성·공유하는 웹 커뮤니티입니다.

**유저픽**은 사용자가 리뷰를 남긴 영화가 모여 있는 목록입니다. 인기 영화에서 심플 리뷰를 작성하면 해당 영화가 유저픽에 올라가고, 리뷰 게시판에서 게시글형 리뷰를 써도 유저픽에 반영됩니다. 이렇게 쌓인 유저픽 목록을 통해 다른 사용자들이 추천받은 영화를 볼 수 있습니다.

---

## 사용한 기술 스택

| 구분 | 기술 |
|------|------|
| **Backend** | Java, Spring Boot, MyBatis, Oracle |
| **Frontend** | React.js |
| **인증** | JWT, BCrypt, Interceptor, AOP |
| **API 연동** | TMDB API |
| **이메일** | SMTP (Gmail) |
| **배포** | AWS EC2 |

---

## 시스템 아키텍처

사용자 → AWS EC2(React SPA) → Spring Boot API(REST API + JWT). 백엔드는 MyBatis·Oracle, BCrypt·TMDB API·로컬 파일 저장·SMTP(Gmail)·Swagger UI와 연동됩니다.

### 테스트 계정

| 구분 | 아이디 | 비밀번호 | 이용 가능 기능 |
|------|--------|----------|----------------|
| **일반 회원** | user01 / user02 | 1234 | 인기 영화·유저픽 조회, 심플/게시글 리뷰 작성·댓글 |
| **관리자** | admin | 1234 | 일반 회원 기능 + 유저픽·리뷰 관리, 리뷰 통계, 회원 관리 |

### 이용 가이드

1. **회원가입·로그인**  
   상단 **회원가입**에서 아이디·비밀번호·닉네임·전화번호·이메일을 입력하고 이메일 인증 후 가입합니다. **로그인** 후 서비스를 이용할 수 있으며, **아이디 찾기**·**비밀번호 찾기**는 이메일로 안내됩니다. 테스트 시에는 위 **테스트 계정**으로 로그인하면 됩니다.

2. **메인 화면**  
   로그인 전에도 **인기 영화 TOP 5**, **유저픽 TOP 5**, **최근 리뷰**를 볼 수 있습니다. **인기 영화 보기** 버튼이나 상단 **인기 영화 / 유저픽 / 리뷰 게시판** 메뉴로 이동할 수 있습니다.

3. **인기 영화**  
   TMDB 기준 인기 영화 목록을 보고, 영화 제목·장르로 검색할 수 있습니다. 영화를 클릭하면 상세 페이지(포스터·줄거리·평점 등)로 이동하며, 로그인한 경우 **해당 영화당 1회** **심플 리뷰**(짧은 소감+평점) 또는 **게시글 리뷰**를 작성할 수 있습니다. 리뷰를 쓰면 해당 영화가 **유저픽**에 올라갑니다.

4. **유저픽**  
   사용자들이 리뷰를 남긴 영화 목록을 제목·장르·상태로 검색해 볼 수 있습니다. 상세 페이지에서는 영화 정보·평가 통계·심플/게시글 리뷰·댓글·대댓글을 확인하고, 본인은 댓글 작성·수정·삭제가 가능합니다. 해당 영화에 리뷰를 안 썼다면 **게시글 리뷰 작성**으로 이동할 수 있습니다.

5. **리뷰 게시판**  
   **게시글형 리뷰**(제목·평점·본문·첨부파일)를 영화 검색 후 작성할 수 있습니다. 목록에서는 영화 제목·장르·리뷰 제목으로 검색하고, 상세에서 **추천**·**댓글·대댓글**을 쓸 수 있습니다. 작성자 본인 또는 **관리자**만 수정·삭제가 가능합니다.

6. **마이페이지**  
   **내정보**에서 닉네임·프로필 이미지 수정, **비밀번호 변경**이 가능합니다. **내 심플 리뷰**·**내 게시글 리뷰**에서 작성 수·평균 평점·장르별 통계를 보고 목록을 검색해 볼 수 있습니다.

7. **관리자**  
   관리자(admin)로 로그인하면 **관리자** 메뉴에서 회원·심플 리뷰·게시글 리뷰·유저픽 목록을 조회·검색·삭제하고 통계를 확인할 수 있습니다. 관리자는 모든 콘텐츠 수정·삭제가 가능합니다.

## 주요 기능

- 회원 가입·로그인(JWT)·회원정보 수정
- 인기 영화·유저픽(추천) 조회·검색
- 심플 리뷰·게시글 리뷰 CRUD·댓글·좋아요
- 관리자: 회원·유저픽·리뷰 관리·통계 조회
